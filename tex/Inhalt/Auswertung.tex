% Ausblick

- Cypher als standardisierte Sprache (Holzschuher und Peinl) kann nur unterstützt werden
- Gremlin wird nicht primär von Neo Technologies, sondern von Aurelius entwickelt
	- Unterstützung aktueller Features ist Community-Arbeit

- Rückwirkung auf Dokumentationen (OrientDB Transaktionen, Titan Kardinalität. Neo4j Anfragebeispiele)
